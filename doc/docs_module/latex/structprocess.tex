\doxysubsection{Detailed Description}
The process that initialized/enabled UMS mechanism. 

This is a node in the \mbox{\hyperlink{structprocess__list}{process\+\_\+list}}. This is a process that initialized/enabled UMS mechanism. Each such process has 3 lists\+:
\begin{DoxyItemize}
\item list of completion lists
\item list of ums threads(schedulers)
\item list of worker threads 
\end{DoxyItemize}
