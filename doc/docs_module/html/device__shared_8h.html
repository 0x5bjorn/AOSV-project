enum yield_reason

Parameters passed by the library and user for worker thread yield function.

Enumerator
FINISH 

The yield reason is FINSIH when worker thread has finished its task

PAUSE 

The yield reason is PAUSE when worker thread hasn't finished its task but for some reason is requested to be blocked/paused until next invokation
